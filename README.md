This project is a simple exploratory data analysis (EDA) of the Titanic dataset using Python, Pandas, Matplotlib and Seaborn.

Goals

Perform basic data cleaning

Explore the structure of the dataset

Calculate descriptive statistics

Visualize key variables such as age and fare

Practice grouping, aggregations, and plotting

Technologies

Python

Pandas

Matplotlib

Seaborn

Google Colab

Example analyses

Mean age by passenger class

Missing data detection

Histogram of passenger ages

Scatter plot: age vs. fare

How to run

Open the .ipynb notebook in Google Colab or Jupyter Notebook.
